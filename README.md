# 🍱 맛있는 급식 - 학교 급식 정보 웹앱

학교 급식 정보를 쉽고 예쁘게 확인할 수 있는 웹 애플리케이션입니다.

## ✨ 주요 기능

- 🏫 **학교 검색**: 실시간 자동완성으로 학교 검색
- 📅 **일별/주간 급식 조회**: 원하는 날짜의 급식 정보 확인
- 🎨 **예쁜 UI**: 직관적이고 아름다운 사용자 인터페이스
- 📱 **모바일 최적화**: 핸드폰에서도 완벽하게 작동
- 💾 **자동 저장**: 기본 학교 설정으로 다음 방문 시 자동 로드
- 🌾 **상세 정보**: 칼로리, 원산지 정보 제공

## 🚀 사용 방법

### 1. 파일 준비
- `optimized.html` 파일을 다운로드합니다
- 웹 브라우저로 파일을 엽니다

### 2. 학교 검색
1. "학교 검색" 입력란에 학교 이름을 입력합니다
2. 자동완성 목록에서 원하는 학교를 선택합니다
3. 선택된 학교가 표시됩니다

### 3. 급식 조회
1. 날짜를 선택합니다
2. "🍱 일별 급식" 버튼을 클릭합니다
3. 급식 정보가 화면에 표시됩니다

### 4. 주간 급식 보기
- "📅 주간 급식" 버튼을 클릭하면 이번 주 전체 급식을 볼 수 있습니다

## ⚙️ 설정

### 기본 학교 설정
1. 우측 상단의 ⚙️ 버튼을 클릭합니다
2. 학교를 검색하고 선택합니다
3. 선택한 학교가 자동으로 기본 학교로 저장됩니다
4. 다음 방문 시 자동으로 해당 학교의 급식이 로드됩니다

### 초기화
- 우측 상단의 🔄 버튼을 클릭하면 모든 설정이 초기화됩니다

## 🖼️ 배경 이미지 설정 (선택사항)

### 방법 1: 로컬 이미지 사용
1. 원하는 배경 이미지를 준비합니다
2. 이미지 파일명을 `lunch-background.png` (또는 `.jpg`)로 변경합니다
3. `optimized.html` 파일과 같은 폴더에 저장합니다
4. 브라우저를 새로고침합니다

### 방법 2: 투명도 조절
`optimized.html` 파일을 텍스트 에디터로 열고 다음 부분을 수정합니다:

```javascript
opacity: 0.15  // 0.1 (더 흐림) ~ 1.0 (선명함)
```

### 방법 3: 이미지 크기 조절
```javascript
backgroundSize: 'cover'  // 'contain' 또는 '100% 100%'로 변경 가능
```

## 📱 모바일 사용

- 모바일 브라우저에서도 완벽하게 작동합니다
- 터치 최적화된 인터페이스
- 반응형 디자인으로 모든 화면 크기 지원

## 🔑 API 키 설정

이 앱은 나이스(NEIS) 오픈 API를 사용합니다.

### API 키 발급 방법
1. https://open.neis.go.kr 접속
2. 회원가입 및 로그인
3. '인증키 신청' 메뉴에서 신청
4. 발급받은 키를 `optimized.html` 파일의 다음 부분에 입력:

```javascript
const NEIS_API_KEY = '여기에_발급받은_키_입력';
```

**참고**: 현재 파일에는 테스트용 API 키가 포함되어 있습니다.

## 💡 팁

### 빠른 사용법
- 자주 사용하는 학교는 설정에서 기본 학교로 저장하세요
- 날짜 선택 후 바로 일별 급식 버튼을 누르면 빠르게 조회됩니다

### 문제 해결

**Q: 급식 정보가 안 나와요**
- A: 학교를 먼저 선택했는지 확인하세요
- A: 날짜가 올바른지 확인하세요 (주말/공휴일은 급식이 없을 수 있습니다)

**Q: 학교 검색이 안 돼요**
- A: 학교 이름을 정확하게 입력했는지 확인하세요
- A: 최소 2글자 이상 입력해야 검색됩니다

**Q: 배경 이미지가 안 보여요**
- A: 이미지 파일명이 `lunch-background.png`인지 확인하세요
- A: 이미지 파일이 `optimized.html`과 같은 폴더에 있는지 확인하세요

**Q: 날짜를 바꿔도 급식이 안 바뀌어요**
- A: 날짜 선택 후 "일별 급식" 버튼을 다시 클릭하세요

## 🎨 커스터마이징

### 색상 변경
`optimized.html` 파일에서 다음 부분을 수정하여 색상을 변경할 수 있습니다:

```javascript
// 그라데이션 색상
from-purple-400 to-pink-400  // 메인 버튼
from-pink-100 via-purple-100 to-blue-100  // 배경

// 텍스트 그라데이션
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### 폰트 변경
현재 'Gowun Dodum' 폰트를 사용 중입니다. 변경하려면:

```css
font-['Gowun_Dodum']  /* 다른 폰트명으로 변경 */
```
